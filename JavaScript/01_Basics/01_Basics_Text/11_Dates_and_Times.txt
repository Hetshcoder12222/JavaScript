2.

    Definition: 


    In JavaScript, the .toString() method when called on a Date object returns a string that represents the Date in a human-readable format.

    Key Points: 

    Converts the Date object to a readable string.
    The format typically looks like:
    "Day Month Date Year HH:MM:SS GMT+Timezone (Time Zone Name)"
    It is automatically called when you try to display a Date object as a string, like in console.log(date).


3.

    Definition: 

    The .toDateString() method is used to convert a JavaScript Date object into a human-readable date string format, without the time portion.

    Return Value: 

    Returns a string representing the date portion of the Date object:

    Key Points: 

    Displays only the date, excluding time.
    Format: "Day Mon Date Year"
    Useful for readable date output.


4.

    Definition: 

    The .toLocaleString() method of a Date object returns a string with a language-sensitive representation of the date and time.
    It allows you to format the date and time according to different locales (languages and regions) and provides options to customize the format.

    locales (optional): A string (like 'en-US', 'hi-IN') or array of locale strings.
    options (optional): An object to customize the date and time format (like { weekday: 'long', year: 'numeric' }).

    Key Points: 

    Locale-sensitive formatting.
    Supports customization (date style, time style, calendar, numbering system, etc.).
    Great for building internationalized applications.

10.

    Definition:

    The .getTime() method is used with JavaScript Date objects to return the number of milliseconds that have passed since January 1, 1970, 00:00:00 UTC (also known as the Unix Epoch).

    Returns: 

    A number representing the time value in milliseconds since the Unix Epoch.


    Key Points: 

    .getTime() is commonly used to compare dates or to measure time intervals.
    Useful when you need the exact time representation of a date.
    The result is the same as Date.now() if called on the current date.
