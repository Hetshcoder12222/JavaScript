    Definition

    The .pop() method is used to remove the last element from an array and return that element. This method changes the length of the array.

    Key Points: 

    Removes the last element from the array.
    Returns the removed element.
    If the array is empty, it returns undefined.

3.

    Definition: 

    The .unshift() method adds one or more elements to the beginning of an array and returns the new length of the array.

    Returns: 

    The new length of the array.

    Key Points: 

    It modifies the original array.
    It is the opposite of .shift(), which removes the first element.
    It is similar to .push(), but adds elements at the start instead of the end.
